- What are your initial impressions of how `Encapsulation` affects your programming?

In this program, encapsulation in each individual class manages its own data and only allows access through methods. For example, the Passenger class stores the passenger’s name, meanwhile the Car class manages the list of passengers inside. Another instance is in the Train class which does not directly handle passengers, but it does interact with cars, which then manage their passengers. By keeping important variables private and using methods like addPassenger() and printManifest(), the program keeps responsibilities organized.
